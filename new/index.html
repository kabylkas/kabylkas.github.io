➡️ Experience

Software Engineer

2021 - Present
Advanced Micro Devices (remotely)

Working at AMD project as a contractor Senior Software Engineer through Luxoft. I develop and maintain the GPU machine-readable ISA documentation, which includes a set of XML files describing AMD's latest GPU ISAs. These specifications cover instructions, encodings, operands, data formats, and human-readable descriptions. I also work on the AMD GPU IsaDecoder API, a C++ tool that allows for easy parsing of these XML files and decoding of instructions and entire shaders. This tool supports multiple AMD architectures, including CDNA™ 3, CDNA™ 2, RDNA™ 3, and RDNA™ 2, and provides developers with an efficient way to interact with and analyze GPU ISAs.

It is open-source! Checkout the project page here: https://gpuopen.com/machine-readable-isa/, and GitHub page here: https://github.com/GPUOpen-Tools/isa_spec_manager
